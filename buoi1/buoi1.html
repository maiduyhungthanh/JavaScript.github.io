Yêu cầu học viên về tự tìm hiểu thêm về

        if/else
        switch/case
        for loop
        Những kiến thức trên tương tự như Java
        
        Làm bài tập sau:
        
        Bài 1: Viết function với tham số đầu vào là mark có giá trị từ 0 -> 100. Kiểm tra giá trị của biến mark và in ra nội dung sau
        
        ”A” nếu mark >= 85
        ”B” nếu 70 <= mark < 85
        ”C” nếu 40 <= mark < 70
        Các trường hợp còn lại in ra “D”
        Bài 2: Viết function có tham số đầu vào là day có giá trị từ 0 -> 6. In ra các ngày trong tuần tương ứng với giá trị day.
        
        Trong đó: (0 - chủ nhật, 1 - thứ 2, ..., 6 - Thứ 7)
        
        Bài 3: Viết function với 2 tham số là min và max. Trả về kết quả là 1 số nguyên được random nằm trong khoảng từ min -> max
        
        Bài tập vòng lặp
        Bài 4: Viết function tính tổng các số lẻ từ 1 -> 100
        
        Bài 5: Viết function tính tổng các số chia hết cho 3 và 5 trong khoảng từ 0 -> 100
